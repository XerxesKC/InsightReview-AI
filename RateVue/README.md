# RateVue - 商家口碑与评价智能管理系统

RateVue 是一个基于 Vue 3 + Vite 开发的现代化评价管理系统，旨在为管理员、商家和普通用户提供全方位的口碑管理与交互体验。系统集成了 AI 智能助手、文档处理及深度数据分析功能。

## 项目特性

- **多角色权限体系**：
  - **管理员 (Admin)**：管控全局，包括用户/商家管理、系统分析、知识库及文档审核工作台。
  - **商家 (Merchant)**：精细化运营，支持内容发布、评价处理、经营分析及智能经营助手。
  - **普通用户 (User)**：发现与互动，支持地图/列表找店、商家详情查看、个人收藏及智能助理咨询。
- **AI 赋能**：
  - **智能助手**：内置大模型驱动的对话助手，提供个性化建议。
  - **Pandas 分析器**：支持通过自然语言进行复杂的数据分析。
  - **文档自动化**：支持智能文档拆分与加工，辅助知识库构建。
- **可视化分析**：基于 ECharts 提供丰富的经营数据看板与舆情分析。
- **响应式 UI**：采用 Element Plus 组件库，打造流畅且美观的交互界面。

## 技术栈

- **核心框架**：Vue 3 (Composition API)
- **构建工具**：Vite
- **状态管理**：Pinia (配合持久化插件)
- **路由管理**：Vue Router
- **UI 组件库**：Element Plus
- **图表库**：ECharts
- **网络请求**：Axios
- **内容解析**：Markdown-it, Highlight.js
- **开发语言**：TypeScript / JavaScript

## 项目结构概览

```
src/
├── api/          # 接口请求定义
├── assets/       # 静态资源 (图片/样式)
├── components/   # 通用/全局组件
├── layouts/      # 页面布局模板 (Admin/Merchant/User)
├── router/       # 路由配置 (分角色路由管理)
├── stores/       # 全局状态管理
├── utils/        # 工具函数
└── views/        # 页面组件 (核心业务代码)
```

##  快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```
